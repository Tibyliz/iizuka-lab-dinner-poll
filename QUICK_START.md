## ⚙️ 基础配置

### 配置1: 设置投票标题 (1分钟)
```
1. 点击 "Title Settings"
2. 输入标题,例如:
   "Iizuka Lab November 2024 Group Dinner Poll"
3. 或点击 "Reset to Default" 使用智能默认标题
4. 点击 "Save Title"
```

### 配置2: 设置价格 (2分钟)
```
1. 点击 "Price Settings"
2. 输入总费用: 10000
3. 设置百分比:
   - 硕士生: 20%
   - 博士生: 30%
   - 教师职工: 50%
4. 查看实时预览
5. 确认总和 = 100%
6. 点击 "Save Settings"
```

## 💰 追踪付款

### 步骤1: 查看每人应付金额
```
表格中的 "Price" 列显示每人需要支付的金额
```

### 步骤2: 标记付款状态
```
1. 收到款后,勾选该人的付款复选框
2. 自动显示绿色的 ✓ 图标
3. 顶部统计显示 "已付款/总参加人数"
```

### 步骤3: 查看未付款成员
```
1. 点击 "Unpaid" 筛选按钮
2. 只显示未付款的成员
3. 方便提醒收款
```

## 🗂️ 管理投票归档

### 保存当前投票
```
1. 活动结束后点击 "Poll Management"
2. 点击 "Save Current Poll to Archive"
3. 输入名称: "November 2024 Dinner"
4. 点击 Save
```

### 开始新投票
```
1. 点击 "Poll Management"
2. 点击 "Start New Poll"
3. 确认操作 (会清除所有回复)
4. 当前回复被清空,设置保持不变
```

### 查看归档
```
1. 点击 "Poll Management"
2. 点击 "View Archived Polls"
3. 看到所有历史投票
4. 可以恢复、导出或删除
```

## 📤 导出数据

### 导出当前投票
```
点击以下任一按钮:
- "Export CSV" - 简单表格
- "Export XLSX" - Excel文件 (推荐)
- "Export PDF" - PDF报告
```

### 导出归档投票
```
1. Poll Management → View Archived Polls
2. 找到要导出的归档
3. 点击 "Export" 按钮
4. 自动下载XLSX文件
```

## 🆘 遇到问题?

### 问题1: 无法登录
```
解决方案:
1. 确认密码正确: iizukalab (默认)
2. 清除浏览器缓存

```

### 问题2: 看不到数据
```
解决方案:
1. 确认是否有提交回复

```

```
解决方案:
确认三个百分比总和正好是 100%
```

```
解决方案:
1. 确认至少有一个"参加"的回复
2. 确认回复中选择了可用日期
3. 刷新页面
```
